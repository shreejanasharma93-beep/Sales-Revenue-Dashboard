[Sales & Revenue Dashboard]

Project Overview

This project is a Sales & Revenue Analysis Dashboard built using Streamlit.

Features

* Total Sales, Revenue, and Profit KPIs
* Sales trends over time
* Top-performing products
* Interactive filters (Region, Category)

Tools Used

* Python
* Pandas
* Streamlit

Dataset

The dataset includes:
* Order Date
* Product
* Category
* Region
* Sales
* Revenue
* Profit

Outcome

This project demonstrates data visualization, KPI tracking, and business insight generation.
